# Node-Express Chatting Server

## Development Installation

In the project's directory, run the following commands:

```bash
# Clone our repo
$ git clone https://github.com/Big-Silver/Node-Chart.git

# Change directory to your app
$ cd Node-Chart

# Install the project
$ npm Install

# Run the project
$ npm run dev

```
Then Visit

```bash
http://localhost:3000
```
